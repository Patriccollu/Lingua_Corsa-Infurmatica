St’infurmazione si trova in parechji ducumenti, secondu à u so cuntenutu :

### [Traduzzione](Traduzzione.md)
L’infurmazione nant’à a traduzzione in lingua corsa d’_OmegaT_ è di u so situ web

### [Memoria](Memoria.zip)
